# Dex Manga Reader

A premium, open-source Android manga reader crafted with modern mobile engineering practices. Powered by the official MangaDex API, the application prioritizes a distraction-free, highly fluid reading experience paired with advanced customization.

## ✨ Key Features

*   **Jetpack Compose & Material 3**: Crafted entirely in Kotlin with responsive layouts, fluid surface transitions, and a gorgeous system-integrated dark theme.
*   **MangaDex API Integration**: Browse popular catalogs, execute fast multi-term queries, and retrieve comprehensive creator and publication details.
*   **Smart Reader Engine**: Continuous scroll navigation with full-screen focus, customizable chapter controls, and page indicator overlays.
*   **Hybrid Publisher Support**: Gracefully matches standard page streams while dynamically redirecting to official websites (like MangaPlus or Viz Media) for externally restricted chapters.
*   **Custom Content Filtering**: Fully adjustable parameters in the master toolbar to switch content modes across Safe, Suggestive, Erotica (18+), and explicitly-marked material.
*   **Persistent Room Database**:
    *   **Favorites & Update Checks**: Keeps local subscriptions of your favorite stories and automatically checks the API in the background to highlight newly published chapters on app open.
    *   **Reading History**: Tracks precisely which chapters and precise page indices you read last, allowing you to instantly resume anywhere.
